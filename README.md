# Engineering Lab Browser Automation (Clawdbot)

This project demonstrates a WORKING browser automation using Clawdbot.

## What it does
- Uses real Chrome browser (NO Playwright)
- Extracts Civil & Mechanical Engineering lab details
- Collects testing machines & lab in-charge info
- Generates structured report
- Sends report automatically via WhatsApp

## Tested
- Windows 10
- Chrome Browser
- Clawdbot CLI

This repo is meant for reproducible testing on a fresh system.
