## System Requirements
- Windows 10/11
- Node.js installed
- Google Chrome installed
- Git installed

## Install Clawdbot
npm install -g clawdbot

## Initial Setup
clawdbot setup
clawdbot onboard

Choose:
- Gateway: Local (Loopback)
- Browser: Chrome Browser Relay
- Channel: WhatsApp

## Start Gateway
clawdbot gateway
